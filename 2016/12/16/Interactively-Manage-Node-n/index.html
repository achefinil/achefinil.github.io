安装 n

如果机器上没有node与npm环境，也可以直接git clone源码安装（需要切换到n目录中执行 make install）。

$ npm install -g n

成功后，Terminal中键入如下命令（会列出已经安装并标记正在使用的版本，可以通过上下键切换要使用的版本，按回车生效）。

$ n
